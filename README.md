# DTEmpire Optimized

An optimization pack for Minecraft, designed to enhance the game while preserving that classic Vanilla feel! Key features include drastically increased FPS, updated HUD and UI elements, mob animations, and quality-of-life improvements. Inspired by Minecraft Dungeons, this pack also introduces new structures and other additions that bring fresh visual appeal and smoother gameplay.
